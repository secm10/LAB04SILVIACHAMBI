¿Cómo elimino una rama creada en git? 
r.- se usa el siguiente comando:
     git branch -d nombre_rama
    $ git branch -d feature/QA

¿Por qué debería de usar la ramificación en proyectos?
R.-las ramificaciones ayudan para poder corregir errores o
a experimentar sin riesgo de estropear el proyecto 
especialmente cuando el proyecto este en produccion.
Usted recibió un ticket [SEM-005] y una vez terminado el 
trabajo usted decide crear una rama para un posterior PR. 
¿Qué nombre pondría a la rama al momento de crearla?
R.- git branch feature/SEM-005
